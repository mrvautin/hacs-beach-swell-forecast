# Beach Swell Forecast

This project provides a forecast for beach swells using various data sources and predictive algorithms.

## Features

Forcasts the next 7 days with well data like: 

- Probability
- SurfMinFt
- SurfMaxFt
- SurfMinM
- SurfMaxM
- SurfPower
- OptimalScore
- HumanRelation
- ForecastDate
